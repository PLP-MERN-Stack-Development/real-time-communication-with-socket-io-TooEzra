# Real-Time Chat App with GIFs & Emojis

A **full-stack, real-time chat application** built with **React + TypeScript**, **Socket.IO**, **Node.js**, and **Express**. Features include:

- Real-time messaging
- Create & join unlimited chat rooms
- Send **GIFs** (powered by GIPHY)
- Send **Emojis** with emoji picker
- JWT-based authentication
- Clean, responsive UI with Tailwind CSS
- Fully TypeScript + ESLint clean
- Deploy-ready (Railway / Render)
